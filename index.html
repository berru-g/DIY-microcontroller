    • Bienvenue dans ce projet Open Source

    Si vous voulez construire votre propre microcontroller, pad, piano en usb midi, pour jouer sur Ableton ou n'importe qu'elle autre software de création musical ou visuel, vous êtes au bon endroit.

    Vous trouverez dans le menu, tout les outils nécessaire pour mener à bien votre projet.

    Tout ces travaux, du code aux plan 3D en passant par le shéma électronique (en cours) on étais réalisé par mes soins pendant le confinement en 2020. Ce projet est OpenSource.

    Noté que le PCB est en cours de création et qu'il manque le régulateur de tension et les condensateurs de découplage pour réduire les pics de tension protégeant la carte et ses composents.

    Le code, le schéma de montage et la 3D sont quand à eux fonctionnel.

    Vous pouvez donc fabriquer ce projet à partir d'un simple Arduino Uno.

    Process

    Tout d'abord vous trouverez le code C++ à executer via l'IDE Arduino ou n'importe qu'elle éditeur prenant en compte Arduino comme VScode.

    Executez le dans Arduino, et récupérez le plan du montage electronique tjr dans le menu.

    Le code est relativement simple, chaque input digital et analog sont indiqué, cela vous permet de vous concentrer sur le coté pratique. Suivant vos besoin vous aurez plus de potar que de touchpad et inversement, à ce moment mettez les lignes du code en commentaire "#" pour optimiser les fonctions de votre hardware.
